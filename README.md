# Unity-Test-Shooter

В файле Bin сборка, Source - сам проект с Unity

Управление: WASD

Взаимодействие с аптечками и дверями: F

Бег: Left Shift

Прыжок: Пробел

Стрельба: левая кнопка мыши

Открыть, закрыть меню и закрыть победное сообщение: Escape

Возможности: реализовано передвижение персонажа, можно взаимодействовать с дверями и аптечками. Аптечки повышают запас здоровья, а двери открываются и закрываются. Игрок может стрелять. Так же есть возможность получать очки за каждого убитого врага. Враги отнимают здоровье при столкновении. 

Условия победы: убить всех врагов (4 двигающихся кубика)

Условия поражения: необходимо дать подойти врагу и не отходить пока враг не снимет всю полоску здоровья.

*Аптечка не возьмется если полоска здоровья полная
